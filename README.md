# Miraculous Kwami Showcase

A fan-made interactive web project built around the Kwami characters from *Miraculous: Tales of Ladybug & Cat Noir*. Each Kwami has its own themed page with a matching color palette, description, video clip, and a discovery popup to navigate between all characters.

## Pages

| Page | Kwami | Theme |
|------|-------|-------|
| index.html | Tikki | Red |
| index2.html | Plagg | Black |
| index3.html | Nooroo | Purple |
| index4.html | Wayzz | Green |
| index5.html | Pollen | Gold |
| index6.html | Trixx | Orange |

## Features

- Per-Kwami themed hero sections with scroll reveal animations
- Discover popup — browse and navigate to any Kwami page
- Watch Live popup — plays a local mp4 clip for each Kwami
- Fully responsive across desktop, tablet, and mobile
- Single shared stylesheet (`main.css`) for all pages

## Built With

- HTML5, CSS3, Vanilla JavaScript
- [ScrollReveal](https://scrollrevealjs.org/)
- [Remix Icon](https://remixicon.com/)
- [Box Icons](https://boxicons.com/)
- Google Fonts — Poppins, Permanent Marker

## Setup

No build tools needed. Just open any `index*.html` file in a browser.

Make sure the following assets are in the same folder as the HTML files:

- `tiki.png`, `plagg.png`, `nooroo.png`, `wayzz.png`, `pollen.png`, `trixx.png`
- `Tikki.mp4`, `Plagg.mp4`, `Nooroo.mp4`, `wayzz.mp4`, `pollen.mp4`, `trixx.mp4`
